private key,foreign key

  1. a primary key is thay unique id for every row in that table.当primary key在其他table被使用时即foreign key。

Design principles

  • Create one table for each entity in your dataset
  • All tablese should have a primary key.
  • The information in that table should depend on the primary key only.
